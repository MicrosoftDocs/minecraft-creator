# Loot Tables Documentation

| Loot Tables | Description |
|:-----|:----------|
| [LootCondition](LootTableComponents/loot_condition.md)| Defines a condition that must be met for a loot table entry or function to apply. |
| [LootTableEntry](LootTableComponents/loot_entry.md)| Defines an entry in a loot table pool that can be selected when the pool rolls. |
| [LootFunction](LootTableComponents/loot_function.md)| Defines a function that modifies a loot table entry when selected. |
| [LootPool](LootTableComponents/loot_pool.md)| Defines a pool of loot entries that can be rolled one or more times. |
| [LootTable](LootTableComponents/loot_table.md)| Defines the loot that can be generated when an entity is killed, a chest is opened, fishing is performed, or other gameplay events occur. |
