# Entity Documentation - minecraft:burns_in_daylight

`minecraft:burns_in_daylight` allows an entity to burn when it's daylight.

## Example

```json
"minecraft:burns_in_daylight": {
}
```

## Vanilla entities examples

### phantom

```json
"minecraft:burns_in_daylight": {
}
```
