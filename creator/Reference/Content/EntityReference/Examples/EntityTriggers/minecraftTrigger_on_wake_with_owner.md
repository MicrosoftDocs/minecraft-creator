# Entity Documentation - minecraft:on_wake_with_owner

A trigger when a mob's tamed onwer wakes up.

|Name       |Default Value |Type |Description |Example Values |
|:----------|:-------------|:----|:-----------|:------------- |

| target | self | String | The target of the event. |  |
