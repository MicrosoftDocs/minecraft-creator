# JSON UI Documentation

| JSON UI | Description |
|:-----|:----------|
| [ui_defs](JsonUiComponents/ui_defs.md)| The _ui_defs.json file that lists all UI screen files that should be loaded by Minecraft. |
| [ui_element](JsonUiComponents/ui_element.md)| A UI element definition in a Minecraft Bedrock UI screen file. |
| [ui_global_variables](JsonUiComponents/ui_global_variables.md)| The _global_variables.json file that defines global variables used across all UI screens in Minecraft. |
| [ui_screen](JsonUiComponents/ui_screen.md)| A UI screen definition file for Minecraft Bedrock Edition. |
