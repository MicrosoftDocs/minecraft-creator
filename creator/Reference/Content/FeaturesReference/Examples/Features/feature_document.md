# Feature Documentation - minecraft:feature_document

A Minecraft Bedrock Edition feature definition document. Features are world generation elements like ore deposits, trees, geodes, and other placeable structures. They define the blocks to place and how to arrange them.


## Feature Document Properties

|Name       |Default Value |Type |Description |Example Values |
|:----------|:-------------|:----|:-----------|:------------- |
| format_version | *not set* | Version | The format version of this feature file. Use '1.13.0' or later. |  |
