# Trade Tables Documentation

| Trade Tables | Description |
|:-----|:----------|
| [TradeTable](TradeTableComponents/trade.md)| Defines a trade table with tiers of trades for villagers or other trading entities. |
| [TradeGroup](TradeTableComponents/tradegroup.md)| A group of trades from which one or more can be randomly selected for a villager to offer. |
| [TradeItem](TradeTableComponents/tradeitem.md)| Defines an item used in a trade, either as input (wants) or output (gives). |
| [TradeItemList](TradeTableComponents/tradeitemlist.md)| Defines a list of alternative items that can satisfy a trade requirement. |
| [TradeItemQuantity](TradeTableComponents/tradeitem_quantity.md)| Defines a random quantity range for a trade item. |
| [TradeSingle](TradeTableComponents/tradesingle.md)| Defines a single trade offered by a villager or wandering trader. |
| [TradeTableData](TradeTableComponents/tradetabledata.md)| Contains internal metadata and type definitions for trade table structures. |
| [TradeTier](TradeTableComponents/tradetier.md)| Defines a tier of trades that becomes available as a villager levels up. |
