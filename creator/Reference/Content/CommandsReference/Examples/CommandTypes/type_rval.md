# Rotation Value

A rotation angle in degrees. Can be absolute or relative.

**Format:** `0-360 or ~offset`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `0` | South (0°) |
| `90` | West (90°) |
| `180` | North (180°) |
| `-90` | East (-90° or 270°) |
| `~` | Keep current rotation |
| `~90` | Rotate 90° from current |
