# Camera Preset

Named camera preset from behavior packs.

**Format:** `namespace:preset_name`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `minecraft:free` | Free-moving camera |
| `minecraft:first_person` | First-person view |
| `minecraft:third_person` | Third-person behind player |
| `minecraft:third_person_front` | Third-person facing player |
