# Equipment Slot Type

Category of inventory or equipment slot.

**Format:** `slot.weapon, slot.armor, slot.inventory, etc.`

## Examples

### Weapon

| Example | Description |
|:--------|:------------|
| `slot.weapon.mainhand` | Main hand slot |
| `slot.weapon.offhand` | Off hand slot |

### Armor

| Example | Description |
|:--------|:------------|
| `slot.armor.head` | Helmet slot |
| `slot.armor.chest` | Chestplate slot |
| `slot.armor.legs` | Leggings slot |
| `slot.armor.feet` | Boots slot |

### Containers

| Example | Description |
|:--------|:------------|
| `slot.inventory` | Player inventory (0-26) |
| `slot.hotbar` | Hotbar (0-8) |
| `slot.enderchest` | Ender chest inventory |

### Mount

| Example | Description |
|:--------|:------------|
| `slot.saddle` | Saddle slot on mount |
| `slot.armor` | Horse armor slot |
| `slot.chest` | Chest slot on donkey/mule |
