# Text String

A text value. May need quotes if it contains spaces.

**Format:** `text or "quoted text"`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `MyName` | Simple text without spaces |
| `"My Name With Spaces"` | Text with spaces (requires quotes) |
