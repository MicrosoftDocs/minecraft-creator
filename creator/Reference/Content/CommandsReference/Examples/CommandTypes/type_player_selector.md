# Player Selector

Targets one or more players. Similar to entity selector but only matches players.

**Format:** `@a, @p, @r, or @s`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `@a` | All players |
| `@p` | Nearest player |
| `@r` | Random player |
| `@s` | Self (if executor is a player) |
