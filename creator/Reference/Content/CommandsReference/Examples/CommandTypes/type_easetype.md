# Easing Type

Animation easing function for camera transitions.

**Format:** `linear, ease_in, ease_out, ease_in_out, etc.`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `linear` | Constant speed |
| `ease_in` | Start slow, accelerate |
| `ease_out` | Start fast, decelerate |
| `ease_in_out` | Smooth start and end |
| `spring` | Bouncy spring effect |
