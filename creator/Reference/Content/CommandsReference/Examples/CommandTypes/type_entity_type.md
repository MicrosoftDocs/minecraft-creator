# Entity Type

An entity type identifier (mob, item, projectile, etc.).

**Format:** `namespace:entity_type or entity_type`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `pig` | Pig mob |
| `minecraft:zombie` | Zombie with namespace |
| `armor_stand` | Armor stand entity |
| `mypack:custom_mob` | Custom entity from addon |
