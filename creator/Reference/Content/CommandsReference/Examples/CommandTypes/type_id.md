# Identifier

A unique identifier string. Usually lowercase with underscores.

**Format:** `lowercase_name`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `my_objective` | Scoreboard objective name |
| `player_score` | A score identifier |
