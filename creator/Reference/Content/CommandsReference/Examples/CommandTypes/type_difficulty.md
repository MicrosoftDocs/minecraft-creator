# Difficulty

The world difficulty level.

**Format:** `peaceful, easy, normal, or hard`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `peaceful` | No hostile mobs, health regenerates quickly |
| `easy` | Hostile mobs deal less damage |
| `normal` | Standard difficulty |
| `hard` | Hostile mobs deal more damage, hunger can kill |

### Shorthands

| Example | Description |
|:--------|:------------|
| `p` | Shorthand for peaceful |
| `e` | Shorthand for easy |
| `n` | Shorthand for normal |
| `h` | Shorthand for hard |

### Numeric

| Example | Description |
|:--------|:------------|
| `0` | Numeric: peaceful |
| `1` | Numeric: easy |
| `2` | Numeric: normal |
| `3` | Numeric: hard |
