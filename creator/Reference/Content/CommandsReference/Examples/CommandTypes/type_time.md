# Time

Time value in ticks or named time.

**Format:** `ticks or day/night/noon/midnight/sunrise/sunset`

## Examples

### Named

| Example | Description |
|:--------|:------------|
| `day` | Start of day (1000 ticks) |
| `night` | Start of night (13000 ticks) |
| `noon` | Midday (6000 ticks) |
| `midnight` | Midnight (18000 ticks) |
| `sunrise` | Sunrise (23000 ticks) |
| `sunset` | Sunset (12000 ticks) |

### Ticks

| Example | Description |
|:--------|:------------|
| `0` | Dawn (tick 0) |
| `6000` | Noon in ticks |
