# Effect

A status effect (potion effect) identifier.

**Format:** `effect_name`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `speed` | Speed boost |
| `strength` | Increased attack damage |
| `invisibility` | Become invisible |
| `night_vision` | See in the dark |
| `regeneration` | Heal over time |
| `resistance` | Reduced damage |
| `fire_resistance` | Immune to fire damage |
| `water_breathing` | Breathe underwater |
| `jump_boost` | Jump higher |
| `slow_falling` | Fall slowly |
