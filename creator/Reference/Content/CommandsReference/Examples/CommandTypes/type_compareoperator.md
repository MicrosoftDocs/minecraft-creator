# Comparison Operator

Operator for comparing values in scoreboard tests.

**Format:** `<, <=, =, >=, or >`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `<` | Less than |
| `<=` | Less than or equal to |
| `=` | Equal to |
| `>=` | Greater than or equal to |
| `>` | Greater than |
