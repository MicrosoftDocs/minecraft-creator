# Message

A text message that will be displayed to players. Can include selector mentions that expand to entity names.

**Format:** `Any text, optionally with @selectors`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `Hello World!` | Simple text message |
| `Welcome, @p!` | Message with nearest player's name |
| `@a has joined the game` | Lists all player names |
