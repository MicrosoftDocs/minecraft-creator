# Enchantment

An enchantment identifier.

**Format:** `enchantment_name`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `sharpness` | Increased melee damage |
| `protection` | Reduced damage from all sources |
| `efficiency` | Faster mining |
| `unbreaking` | Increased durability |
| `fortune` | Increased block drops |
| `silk_touch` | Mine blocks directly |
| `mending` | Repair with XP |
| `looting` | Increased mob drops |
