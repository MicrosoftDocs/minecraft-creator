# Block ID

A block type identifier. Can include namespace or use short form for vanilla blocks.

**Format:** `namespace:block_name or block_name`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `stone` | Stone block (vanilla) |
| `minecraft:diamond_block` | Diamond block with namespace |
| `oak_planks` | Oak planks |
| `mypack:custom_block` | Custom block from addon |
