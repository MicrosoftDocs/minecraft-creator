# Scoreboard Operation

Mathematical operation for scoreboard calculations.

**Format:** `+=, -=, *=, /=, %=, =, <, >, or ><`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `+=` | Add value |
| `-=` | Subtract value |
| `*=` | Multiply by value |
| `/=` | Divide by value (integer division) |
| `%=` | Modulo (remainder) |
| `=` | Set to value |
| `<` | Set to minimum of both values |
| `>` | Set to maximum of both values |
| `><` | Swap values between targets |
