# Function Path

Path to a function file within a behavior pack. Uses namespace:path format.

**Format:** `namespace:path/to/function`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `mypack:setup` | Function 'setup' in namespace 'mypack' |
| `mypack:utils/helper` | Function in a subfolder |
| `tick` | Function in default namespace |
