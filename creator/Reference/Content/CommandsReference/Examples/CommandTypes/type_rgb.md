# RGB Color

Color value as red, green, blue components.

**Format:** `0-255 for each component`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `255 0 0` | Red |
| `0 255 0` | Green |
| `0 0 255` | Blue |
| `255 255 255` | White |
| `0 0 0` | Black |
