# Block Position

Block coordinates as integers. Does not accept decimal values.

**Format:** `x y z or ~ ~ ~`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `~ ~ ~` | Current block position |
| `0 64 0` | Absolute block coordinates |
