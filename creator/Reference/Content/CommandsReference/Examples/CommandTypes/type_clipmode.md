# Clip Mode

How to handle entity clipping during camera/spectator mode.

**Format:** `player, camera, or default`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `player` | Player clipping behavior |
| `camera` | Camera clipping behavior |
