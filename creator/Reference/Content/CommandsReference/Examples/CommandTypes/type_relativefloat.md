# Relative Number

A number that can be relative to current value using ~.

**Format:** `value or ~offset`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `10` | Absolute value of 10 |
| `~5` | Add 5 to current value |
| `~-3` | Subtract 3 from current value |
| `~` | Keep current value (same as ~0) |
