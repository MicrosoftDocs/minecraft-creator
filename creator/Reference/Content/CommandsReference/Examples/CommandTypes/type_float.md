# Decimal Number

A number that can include decimals.

**Format:** `0, 1.5, -3.14, etc.`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `0` | Zero |
| `0.5` | Half |
| `1.5` | One and a half |
| `-0.25` | Negative quarter |
