# Item ID

An item type identifier.

**Format:** `namespace:item_name or item_name`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `diamond` | Diamond item |
| `minecraft:iron_sword` | Iron sword with namespace |
| `golden_apple` | Golden apple |
