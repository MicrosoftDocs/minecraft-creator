# Scoreboard Objective

Name of a scoreboard objective.

**Format:** `alphanumeric with underscores, max 16 characters`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `kills` | Track player kills |
| `deaths` | Track player deaths |
| `score` | General score tracking |
| `money` | Currency tracking |
