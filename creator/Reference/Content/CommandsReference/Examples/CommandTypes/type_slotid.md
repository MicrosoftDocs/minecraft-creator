# Slot Number

Numeric slot index within a slot type.

**Format:** `0-26 for inventory, 0-8 for hotbar`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `0` | First slot in the container |
| `8` | Last hotbar slot |
| `26` | Last inventory slot |
