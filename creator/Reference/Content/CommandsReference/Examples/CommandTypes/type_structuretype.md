# Structure Type

A structure identifier for locate command.

**Format:** `structure_name`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `village` | Village structure |
| `stronghold` | Stronghold |
| `mineshaft` | Abandoned mineshaft |
| `monument` | Ocean monument |
| `fortress` | Nether fortress |
| `mansion` | Woodland mansion |
