# Suffixed Value

A value with a unit suffix.

**Format:** `number with suffix (e.g., 5s, 10t)`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `5s` | 5 seconds |
| `100t` | 100 ticks |
| `1d` | 1 day (in-game) |
