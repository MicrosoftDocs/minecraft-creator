# Integer Range

A range of integers, or a single integer.

**Format:** `min..max, ..max, min.., or single value`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `1..10` | Between 1 and 10 inclusive |
| `..5` | 5 or less |
| `10..` | 10 or more |
| `5` | Exactly 5 |
