# Weather Type

The weather condition.

**Format:** `clear, rain, or thunder`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `clear` | Clear skies, sunny |
| `rain` | Raining (snowing in cold biomes) |
| `thunder` | Thunderstorm with lightning |
