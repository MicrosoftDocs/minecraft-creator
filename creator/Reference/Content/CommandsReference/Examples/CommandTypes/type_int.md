# Integer

A whole number (no decimals).

**Format:** `..., -2, -1, 0, 1, 2, ...`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `0` | Zero |
| `64` | Positive integer |
| `-10` | Negative integer |
