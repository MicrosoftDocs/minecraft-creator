# Game Mode

The player's game mode.

**Format:** `survival, creative, adventure, or spectator`

## Examples

### Values

| Example | Description |
|:--------|:------------|
| `survival` | Survival mode - gather resources, hunger, health |
| `creative` | Creative mode - unlimited resources, flight, invincible |
| `adventure` | Adventure mode - can't break/place blocks without tools |
| `spectator` | Spectator mode - fly through blocks, invisible |

### Shorthands

| Example | Description |
|:--------|:------------|
| `s` | Shorthand for survival |
| `c` | Shorthand for creative |
| `a` | Shorthand for adventure |

### Numeric

| Example | Description |
|:--------|:------------|
| `0` | Numeric: survival |
| `1` | Numeric: creative |
| `2` | Numeric: adventure |
