# `/packstack` Command

Prints client or server pack stack to chat.

| | |
|:---|:---|
| **Permission Level** | Any |
| **Requires Cheats** | No |

## Syntax Overview

`/packstack <stackType: stacktype> [verbose: verbose] [exclude-vanilla: exclude-vanilla]`

## Usage

`/packstack <stackType: stacktype> [verbose: verbose] [exclude-vanilla: exclude-vanilla]`

| Argument | Type | Required | Description |
|:---------|:-----|:---------|:------------|
| stackType | stacktype | Required |  |
| verbose | verbose | Optional |  |
| exclude-vanilla | exclude-vanilla | Optional |  |
