# `/stop` Command

Stops the server. This command is for use on a dedicated server only.

| | |
|:---|:---|
| **Permission Level** | Owner |
| **Requires Cheats** | Yes |

## Syntax Overview

`/stop`

## Usage

`/stop`

Stop the server.
