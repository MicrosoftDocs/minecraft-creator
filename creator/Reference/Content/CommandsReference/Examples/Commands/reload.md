# `/reload` Command

Reloads all function and script files from all behavior packs.

| | |
|:---|:---|
| **Permission Level** | Admin |
| **Requires Cheats** | Yes |

## Syntax Overview

`/reload [all: reload_all]`

## Usage

`/reload [all: reload_all]`

Reload all function and script files.

| Argument | Type | Required | Description |
|:---------|:-----|:---------|:------------|
| all | reload_all | Optional |  |
