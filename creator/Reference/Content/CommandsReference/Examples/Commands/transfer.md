# `/transfer` Command

Transfers a player to another server.

| | |
|:---|:---|
| **Permission Level** | Owner |
| **Requires Cheats** | Yes |

## Syntax Overview

`/transfer <pfidOrMSA: id> <server: id> [port: int]`

## Usage

`/transfer <pfidOrMSA: id> <server: id> [port: int]`

| Argument | Type | Required | Description |

| server | [Identifier](../CommandTypes/type_id.md) | Required |  |
