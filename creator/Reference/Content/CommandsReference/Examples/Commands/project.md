# `/project` Command

This command contains additional tools for managing an Editor project. This command is available in Editor mode only.

| | |
|:---|:---|
| **Permission Level** | Game Directors |
| **Requires Cheats** | No |

## Syntax Overview

`/project <subcommand: subcommandexport> <exportType: exporttypes>`

## Usage

`/project <subcommand: subcommandexport> <exportType: exporttypes>`

| Argument | Type | Required | Description |
|:---------|:-----|:---------|:------------|
| subcommand | subcommandexport | Required | Signifies the start of an export-subcommand operation. |
| exportType | exporttypes | Required |  |

## Arguments Reference

| Argument | Type | Description |
|:---------|:-----|:------------|
| exportType | exporttypes |  |
| subcommand | subcommandexport | Signifies the start of an export-subcommand operation. |
