# `/reloadconfig` Command

Reloads configuration files relating to variables, secrets, permissions, etc.

| | |
|:---|:---|
| **Permission Level** | Owner |
| **Requires Cheats** | Yes |

## Syntax Overview

`/reloadconfig`

## Usage

`/reloadconfig`
