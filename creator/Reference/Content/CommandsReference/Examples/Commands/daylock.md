# `/daylock` Command

Locks and unlocks the day-night cycle.

**Aliases:** `/alwaysday`

| | |
|:---|:---|
| **Permission Level** | Game Directors |
| **Requires Cheats** | Yes |

## Syntax Overview

`/daylock [lock: Boolean]`

## Usage

`/daylock [lock: Boolean]`

Lock daylight with an optional `Boolean`.

| Argument | Type | Required | Description |
|:---------|:-----|:---------|:------------|
| lock | [Boolean](../CommandTypes/type_boolean.md) | Optional | Default `true`. An `enum` of default `true` which locks daytime or `false`. |

## Arguments Reference

| Argument | Type | Description |
|:---------|:-----|:------------|
| lock | [Boolean](../CommandTypes/type_boolean.md) | Default `true`. An `enum` of default `true` which locks daytime or `false`. |
