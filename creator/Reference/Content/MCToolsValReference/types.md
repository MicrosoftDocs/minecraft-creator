# MCTools Validation Rules Documentation - minecraft:types

Aggregates information about all content types defined in the project including entities, blocks, items, and other custom definitions.


## Content Types Validation Rules


### TYPES101: Type Summary

**Rule ID:** `TYPES101`

**Severity:** Info

**What it checks:** Summary of all custom content types (entities, blocks, items, etc.) defined across behavior and resource packs in the project.

**Technical details:** Data contains the type identifier and definition summary.
