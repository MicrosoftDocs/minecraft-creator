# Client Biome Documentation - minecraft:precipitation

Describes the visuals for a biome's precipitation. Biomes without this component will have default values. At most one precipitation type can be set for a biome.


## Client Biome Precipitation Properties

|Name       |Default Value |Type |Description |Example Values |
|:----------|:-------------|:----|:-----------|:------------- |
| ash | *not set* | Decimal number | Density of ash precipitation visuals. Used in the soul sand valley biome. |  | 
| blue_spores | *not set* | Decimal number | Density of blue spore precipitation visuals. Used in the warped forest biome. |  | 
| red_spores | *not set* | Decimal number | Density of red spore precipitation visuals. Used in the crimson forest biome. |  | 
| white_ash | *not set* | Decimal number | Density of white ash precipitation visuals. Used in the basalt deltas biome. |  |
